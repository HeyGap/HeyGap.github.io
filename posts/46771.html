内核编译

linux kernel pub

首先使用 file bzImage 查看使用的 kernel 版本，如下所示，本题目使用的 kernel 版本为 4.4.72

使用 wget https://www.kernel.org/pub/linux/kernel/v4.x/linux-4.4.72.tar.gz 在官网下载含有漏洞的对应版本的 linux kernel 源码

使用 strings bzImage | grep gcc 查看使用的编译器

可以看到我们需要gcc5，首先安装相关依赖

然后去 各版本的 gcc 源码库 下载 gcc5

修改 linux/.config 文件，使其满足我们的需求，我们一般需要启动下面的配置来确保 vmlinux 支持调试信息。在完成一切配置修改后，使用 make olddefconfig && make -j$(nproc) CC=gcc-5 编译即可
