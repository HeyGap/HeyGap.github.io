在移动硬盘中用VirtualBox安装用于Pwn的Kali_linux

发表于 2023-07-17 分类于 Tools 阅读次数：
之前安装过好多次linuxVM，感觉都不满意，这次记录一下

0xFF 说在前面

安装过程中体验了一下清华源的.7z包，点击即送kalilinux，比笔者下面讲的方法方便太多啦，如果读者嫌麻烦的话可以用着清华源的包

0x00 安装过程

  1. 打开virtualbox(VB去Oracle VM VirtualBox下载即可)，点击新建，选择合适的文件存放位置和版本信息，点击下一步（如果是安装kali的话记得把[版本]改为Debian(64-bit)） [内存大小]32位一般1024，64位一般2048就够了。我怕我虚拟机内存不够，所以开了4096
  2. 创建虚拟硬盘，[文件大小]一般20G够用，我因为后面可能要用好长时间，于是开了64GB
  3. 选中刚才创建的虚拟机，进行一些设置的更改 ①设置-->常规-->高级-->共享粘贴板、拖放-->双向 ②系统-->处理器-->CPU=2 || 扩展特性勾选
  4. 下载kali linux镜像我这里用的是清华源，国内下载速度快一些。当然也可以去kali官网进行下载
  5. 选择你刚才下载的盘片
  6. 更改桥接网卡
  7. 双击启动虚拟机，选择'Graphical Install'进行图形化界面安装，然后一路按自己需求配置就好啦。 选择语言-->地区-->键盘配置-->等待安装就好，后续要求“配置网络”的话，只需要按照他显示的内容点击continue就好，不需要修改 这里使用整个磁盘，就是你之前分配的磁盘大小，然后就按照他的默认配置一路走下去就可以进入安装了，安装过程可能有些慢 这里选择下面这一行即可
