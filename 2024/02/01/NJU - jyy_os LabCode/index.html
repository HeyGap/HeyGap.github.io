[M1] pstree

本次实验花了我两个下午...自己的 coding 能力实在是太弱了，不过正好也借这次机会弄清楚了以下几个问题:

  1. why we choose typedef  how we use typedef

  2. how to build a queue（虽然代码没用到队列）

  3. how to build a tree and how to traverse it

  4. how to fetch the INFO of processes -> how to traverse the directory -> how to read file in a formatted way

  5. how to parse the cmdline args

  6. (the most important) how to debug a exefile with gdb

[L0] amgame

太懒了，稍微做一做就不想做了 ;（

拿一下午做个 wasd 移动方块的程序就撤了

一开始还是在用平时常用的 kali，但是发现 ubuntu 更有性价比，于是开了 wsl ubuntu 20.04.06 LTS 继续做了

我这边git pull origin L0不太能正常工作，所以我选择直接去 github 对应 repo 的 branch 直接下 zip 然后解压 😭

然后直接在 /L0/amgame 目录下make

STFW 解决报错以后进入 build 目录，用 qemu-system-x86_64 -drive format=raw,file=amgame-x86_64-qemu 运行，弹出来的新窗口就是我们自己的 OS 啦🙋‍
