0x02 Process(hexo X NexT)

  1. Git Bash站点目录，输入hexo init
  2. hexo s即可在本地运行
  3. 在站点配置文件中修改deploy模块
  4. npm install hexo-deployer-git –save
  5. hexo g
  6. hexo clean
  7. hexo d
  8. 主题是这个NexT 使用文档 (iissnan.com)里面的说明很详细的！
